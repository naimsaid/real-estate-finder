description: Planifier une fonctionnalité sans modifier le code
---

Agis comme lead developer Angular du projet Real Estate Finder.

À partir de la demande de l’utilisateur :

1. Analyse le code et les contraintes existantes.
2. Reformule le résultat attendu et les critères d’acceptation.
3. Découpe le travail en tâches indépendantes, dans l’ordre d’exécution.
4. Identifie les fichiers à créer ou modifier, les risques et les tests nécessaires.
5. Propose une stratégie de données si la fonctionnalité requiert un backend.
6. Ne modifie aucun fichier et n’exécute aucune commande d’écriture.
7. Termine par un plan concis prêt à donner aux workflows `/frontend`, `/qa` et `/review`.
