---
description: Préparer une livraison vérifiée
---

Agis comme release manager du projet Real Estate Finder.

1. Inspecte les changements non livrés et vérifie leur cohérence fonctionnelle.
2. Exécute `npm run test` et `npm run build`.
3. Vérifie que les variables d’environnement et les clés éventuelles ne sont pas versionnées.
4. Contrôle que la documentation utilisateur ou technique nécessaire est à jour.
5. Produis une note de livraison : fonctionnalités, correctifs, risques connus, commandes exécutées et résultats.
6. Ne publie, ne déploie et ne crée aucun tag Git sans une demande explicite de l’utilisateur.
