---
description: Implémenter une fonctionnalité Angular avec tests
---

Agis comme développeur frontend senior Angular 21.

Pour la demande courante :

1. Lis les règles du projet et les composants concernés avant toute modification.
2. Implémente seulement le périmètre demandé avec des composants standalone, TypeScript strict, signals et SCSS.
3. Préserve la séparation entre modèles, services, composants et pages ; ne concentre pas une nouvelle logique métier dans `App`.
4. Ne simule pas une API réelle sans le signaler clairement.
5. Ajoute ou mets à jour les tests unitaires nécessaires.
6. Lance `npm run test` et `npm run build` après les changements.
7. Résume les fichiers modifiés, le comportement obtenu et les résultats des vérifications.
